## Déploiement sur GitHub Pages

1. Créez un dépôt GitHub (ex: `francois-eyden-mariage`)
2. Uploadez tous ces fichiers **en conservant la structure de dossiers**
3. Allez dans **Settings → Pages**
4. Source : `Deploy from a branch` → `main` → `/ (root)`
5. Votre site sera disponible à `https://votrepseudo.github.io/francois-eyden-mariage`

## Personnalisation

- **Section Lieu** : Remplacez `VOTRE_URL_PHOTO_ICI` dans le CSS de la section `#lieu`
- **Date** : Cherchez `2027-07-01` pour changer la date du compte à rebours
- **Photos** : Remplacez les fichiers dans `/images/` par vos propres photos (gardez les mêmes noms)
